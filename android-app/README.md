# 安卓通知自动记账 App

这个 Android 项目用于自动识别微信和支付宝的支付通知。授权通知使用权后，App 会在本机读取通知栏文本，提取金额和商户，自动分类并记录到本机。

## 工作方式

- 监听包名：`com.tencent.mm`、`com.eg.android.AlipayGphone`
- 只处理通知文本里的支出/付款/消费金额
- 跳过退款、到账、收入等通知
- 数据保存在手机本机，不上传
- 可切换查看历史月份、年度账单和分类汇总
- 每月支出超过 `1500` 元时发送系统通知提醒
- 支持导入微信/支付宝历史 CSV 或 TXT 账单
- 支持从文件管理器、微信或支付宝的“更多打开方式/分享”直接打开账单到本 App

## 构建

1. 安装 Android Studio。
2. 打开 `android-app` 文件夹。
3. 等待 Gradle 同步完成。
4. 点击 Run 安装到安卓手机。

也可以在 GitHub 仓库的 Actions 页面运行 `Build Android APK`，完成后下载 `moneyorganizationapp-debug-apk`。

## 手机上开启

1. 打开 App。
2. 点击“导入以前的支付记录”，选择微信或支付宝导出的 CSV/TXT 历史账单。
3. 点击“去开启”。
4. 在系统的“通知使用权”页面里开启“支出管家通知识别”。
5. 如果系统询问通知权限，也允许“支出管家”发送通知。
6. 之后微信/支付宝出现支付成功通知时，App 会自动记录。

## 更快导入历史账单

在手机文件管理器、微信文件或支付宝账单文件里：

1. 点账单文件。
2. 选择“用其他应用打开”“更多打开方式”或“分享”。
3. 选择“支出管家”。
4. App 会自动识别并导入。

如果 App 打开后提示“没有收到文件”，说明当前入口没有把文件传给 App，请换“分享”入口再选“支出管家”。

## 页面功能

- 顶部可切换“上个月 / 下个月”
- 本月或历史月份支出总额
- 每月 1500 元红线进度条
- 年度账单和各月汇总
- 分类汇总
- 最近支出记录
- 历史账单文件导入
- 从其他 App 打开/分享账单文件导入

## 注意

- 必须让微信/支付宝显示支付通知。
- 不同手机厂商的通知文案可能不同，第一版会覆盖常见通知格式。
- 如果某类通知没识别，把通知文字发给我，可以继续补解析规则。
- iPhone 不支持这种通知读取方式。
