# 月度支出管家

一个离线可用的微信 / 支付宝支出整理工具。打开 `index.html` 后导入账单文件，工具会自动识别支出、跳过收入和退款，按月份汇总并按类别展示。

## 使用方式

1. 在浏览器里打开 `index.html`。
2. 点击“导入账单”，选择微信或支付宝导出的 `.csv` / `.txt` 文件。
3. 手机上不方便选文件时，点击“粘贴账单”，把账单表格内容粘贴进去识别。
4. 使用月份、分类和搜索框查看当月支出。
5. 如果自动分类不准确，可以在明细表里直接改分类。

## 手机上使用

这是一个 PWA 小应用。手机浏览器访问后，可以通过浏览器菜单选择“添加到主屏幕”，之后就能像 App 一样打开。

PWA 安装和离线缓存通常需要通过本地服务器或 HTTPS 访问。部署到 GitHub Pages 后，手机可以直接访问：

```text
https://fjy934825-jpg.github.io/moneyorganizationapp/
```

电脑调试时可以在本目录运行：

```powershell
python -m http.server 8080
```

然后在浏览器访问 `http://localhost:8080`。如果要在手机上用，建议把这个文件夹部署到支持 HTTPS 的静态网站，或让手机访问同一局域网里的电脑地址。

## 微信记账助手

如果不想在手机上导入或粘贴账单，可以运行 `wechat-bot` 本地助手。它支持把文字账单、微信/支付宝 CSV 或 TXT 文件发给登录的微信号，自动识别并写入本地账本。

```powershell
cd wechat-bot
npm install
npm run login
npm run start
```

启动后访问：

```text
http://localhost:8787
```

这个地址打开的网页会读取微信助手写入的同一份数据。

## 安卓自动记账

如果使用安卓手机，可以打开 `android-app` 项目，用 Android Studio 构建安装。安装后在系统里开启“通知使用权”，App 会自动识别微信和支付宝的支付通知并记录到本机。App 支持历史月份、年度账单、分类汇总，以及每月 1500 元红线提醒。

也可以在 GitHub 的 Actions 页面运行 `Build Android APK` 自动打包，完成后下载 `moneyorganizationapp-debug-apk`。

详见：

```text
android-app/README.md
```

## 支持能力

- 支持微信支付账单、支付宝账单常见导出格式。
- 支持 UTF-8 和 GB18030 编码账单。
- 自动分类：餐饮、购物、学习、交通、居住、娱乐、医疗、人情、通讯、其他。
- 数据保存在浏览器本地 `localStorage`，不会上传。
- 支持导出当前本地记录为 JSON。
- 支持 PWA：可添加到手机主屏幕，并缓存页面用于离线打开。

## 注意

微信和支付宝的导出模板可能会随版本变化。如果导入后没有识别到记录，可以把账单表头发给我，我可以继续扩展解析规则。
