# 月度支出管家

一个离线可用的微信 / 支付宝支出整理工具。打开 `index.html` 后导入账单文件，工具会自动识别支出、跳过收入和退款，按月份汇总并按类别展示。

## 使用方式

1. 在浏览器里打开 `index.html`。
2. 点击“导入账单”，选择微信或支付宝导出的 `.csv` / `.txt` 文件。
3. 使用月份、分类和搜索框查看当月支出。
4. 如果自动分类不准确，可以在明细表里直接改分类。

## 手机上使用

这是一个 PWA 小应用。手机浏览器访问后，可以通过浏览器菜单选择“添加到主屏幕”，之后就能像 App 一样打开。

PWA 安装和离线缓存通常需要通过本地服务器或 HTTPS 访问。电脑调试时可以在本目录运行：

```powershell
python -m http.server 8080
```

然后在浏览器访问 `http://localhost:8080`。如果要在手机上用，建议把这个文件夹部署到支持 HTTPS 的静态网站，或让手机访问同一局域网里的电脑地址。

## 支持能力

- 支持微信支付账单、支付宝账单常见导出格式。
- 支持 UTF-8 和 GB18030 编码账单。
- 自动分类：餐饮、购物、学习、交通、居住、娱乐、医疗、人情、通讯、其他。
- 数据保存在浏览器本地 `localStorage`，不会上传。
- 支持导出当前本地记录为 JSON。
- 支持 PWA：可添加到手机主屏幕，并缓存页面用于离线打开。

## 注意

微信和支付宝的导出模板可能会随版本变化。如果导入后没有识别到记录，可以把账单表头发给我，我可以继续扩展解析规则。
